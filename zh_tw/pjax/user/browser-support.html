登入故障

如果存在帳戶登入問題，請確認您的瀏覽器cookie 是否啟用。在登入時域期間，本網站使用cookie 在您的電腦上（以加密格式）儲存您的登入ID 和密碼。

要啟用cookie，請按下面符合您瀏覽器版本的指示說明進行操作。

Binary (C.R.) S.A.

Chrome 40+

  • 前往"Chrome菜單(自訂與管理Google Chrome瀏覽器)" 下拉列表並選擇設定。
  • 點選顯示高階設定標籤。
  • 點選隱私 菜單下方的內容設定按鈕。
  • 選擇允許設定本地資料（推薦）。
  • 點選完成 按鈕儲存您的設定。
Binary (IOM) Ltd

Mozilla Firefox 40+

  • 打開選單並選取最愛。
  • 點選 隱私 標籤。
  • 設定"Firefox 將"記住"您的操作記錄"。
  • 點選OK按鈕儲存您的設定。
Binary (Europe) Ltd

Safari 5.x+

  • 前往 Safari 選單，然後選取最愛。
  • 點選 私隱標籤。
  • 將"阻止 cookies:" 設定為 永不。
  • 點選右上方的X按鈕，儲存並關閉。
Binary (C.R.) S.A.

Opera 30+

  • 前往工具 選單並選擇偏好。
  • 點選進階 標籤。
  • 從左側欄選擇Cookies。
  • 選擇接受cookies。
  • 點選OK 按鈕儲存並關閉。
Binary Ltd

Microsoft Internet Explorer 11+

  • 進入工具 選單並選擇網際網路選項。
  • 點選隱私 標籤。
  • 點擊 進階 按鈕。
  • 選擇覆寫自動cookie處理。
  • 選擇接受 第一方Cookies 和第三方Cookies。
  • 點選OK按鈕儲存並關閉。
Binary Ltd

Microsoft Edge 12+

  • 前往更多操作選單並選取 設定。
  • 在進階設定標籤下，點選檢視進階設定按鈕。
  • 在Cookies分區下，選取 "不可阻止 cookies"選項。
  • 選單外任意點選儲存並關閉。
