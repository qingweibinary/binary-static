登录故障

如果存在账户登录问题，请确认您是否已启用浏览器 cookie。在登录期间，本网站使用 cookie 在您的计算机上（以加密格式）存储您的登录 ID 和密码。

要启用 cookie，请根据您的浏览器版本选择对应的指示说明，并进行操作。

Binary (C.R.) S.A.

Chrome 40+

  • 进入"Chrome菜单(个性化设置谷歌Chrome浏览器)" 下拉列表并选择 设置。
  • 点击显示高级设置标签。
  • 点击 隐私菜单下面的内容设置键。
  • 选择 允许设置本地数据（推荐）。
  • 点击 完成 键保存您的设置。
Binary (IOM) Ltd

Mozilla Firefox 40+

  • 打开菜单并选择 偏好。
  • 点击 隐私 标签。
  • 设置 "Firefox 将l" 记住 "您的操作历史"。
  • 点击 OK 键保存您的设置。
Binary (Europe) Ltd

Safari 5.x+

  • 进入 Safari 菜单，然后选择偏好。
  • 点击 私隐选项卡。
  • 将"阻止 cookies:" 设置为 永不。
  • 点击右上方的X 按钮，保存并关闭。
Binary (C.R.) S.A.

Opera 30+

  • 进入 工具 菜单并选择 偏好。
  • 点击高级 标签。
  • 从左侧栏选择Cookies。
  • 选择接受cookies。
  • 点击 OK 键保存并关闭。
Binary Ltd

Microsoft Internet Explorer 11+

  • 进入 工具 菜单并选择互联网选项。
  • 点击 隐私 标签。
  • 点击 高级 键。
  • 选择 覆盖自动cookie处理。
  • 选择 接受 第一方 Cookies 和第三方Cookies。
  • 点击 OK键保存并关闭。
Binary Ltd

Microsoft Edge 12+

  • 进入 更多操作 菜单并选择 设置。
  • 在高级设置 标签下，点击查看高级设置按钮。
  • 在Cookies分区下，选择 "不阻止 cookies"。
  • 在菜单外任意点击，保存并关闭。
