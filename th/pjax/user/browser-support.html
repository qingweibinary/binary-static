ปัญหาการเข้าสู่ระบบ

หากท่านไม่สามารถเข้าใช้งานบัญชีของท่าน โปรดตรวจสอบคุกกี้ของเบราเซอร์ท่าน เว็บไซต์ของเราใช้คุกกี้ที่เก็บอยู่ในบัญชีผู้ใช้งานและรหัสผ่านของท่าน (ที่มีการเข้ารหัส) ในเครื่องคอมพิวเตอร์ของท่านในขณะที่ท่านใช้งานระบบซื้อขาย

เพื่อติดตั้งคุกกี้ โปรดปฏิบัติตามขั้นตอนข้างล่างสำหรับเวอร์ชันของเบราเซอร์ที่ท่านใช้งาน

Binary (C.R.) S.A.

Chrome 40+

  • ไปที่ "เมนู Chrome (Customize and control Google Chrome)" มีรายการตัวเลือก และให้เลือก Settings
  • คลิกที่ Show advanced settings
  • คลิกที่ปุ่ม Content settings ที่อยู่ด้านล่างของเมนู Privacy
  • เลือก Allow local data to be set (recommended)
  • คลิกที่ปุ่ม Done เพื่อบันทึกการตั้งค่า
Binary (IOM) Ltd

Mozilla Firefox 40+

  • เปิดเมนู และเลือก Preferences
  • ไปที่แท็บ Privacy
  • ตั้งค่า "Firefox will" ให้เป็น "Remember history"
  • คลิกที่ปุ่ม OK เพื่อบันทึกการตั้งค่า
Binary (Europe) Ltd

Safari 5.x+

  • ไปที่เมนู Safari และเลือก Preferences
  • คลิกที่แท็บ Privacy
  • ตั้งค่า"Block cookies:" เป็น Never
  • คลิกที่ปุ่ม X (กากบาท) ตรงมุมบนด้านขวาเพื่อบันทึกค่าและปิด
Binary (C.R.) S.A.

Opera 30+

  • ไปที่เมนู Tools และเลือก Preferences
  • คลิกที่ Advanced
  • เลือก Cookies จากด้านซ้ายมือ
  • เลือก Accept cookies
  • คลิกที่ปุ่ม OK เพื่อบันทึกและปิด
Binary Ltd

Microsoft Internet Explorer 11+

  • ไปที่เมนู Tools และเลือก Internet Options
  • คลิกที่ Privacy แท็บ
  • คลิกที่ปุ่ม Advanced
  • เลือก Override automatic cookie handling
  • เลือก Accept ทั้ง First-party Cookies และ Third-party Cookies
  • คลิกที่ปุ่ม OK เพื่อบันทึกและปิด
Binary Ltd

Microsoft Edge 12+

  • ไปที่เมนู More actions และเลือก Settings
  • บริเวณข้างล่าง เลือกที่Advanced Settings, คลิกปุ่ม View advanced settings
  • บริเวณข้างล่าง ส่วนที่เป็น Cookies เลือกค่า "Don't block cookies"
  • คลิกพื้นที่ข้างนอกของเมนูเพื่อบันทึกและปิด
